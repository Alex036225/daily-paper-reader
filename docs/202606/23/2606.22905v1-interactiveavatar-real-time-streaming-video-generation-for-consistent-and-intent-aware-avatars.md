## 摘要
近期基于扩散模型的研究已实现音频驱动的逼真化身实时流式生成。然而，现有方法难以保持视觉时序一致性，且在复杂交互流场景中无法明确感知用户意图。为应对这些挑战，我们提出 InteractiveAvatar，一个支持视觉一致性化身视频生成与意图感知交互的实时无限流视频生成框架。通过自回归蒸馏，InteractiveAvatar 能在任意长时间内实时流式生成人类化身。针对视觉一致性，我们引入长短时视觉记忆（LSVM）机制，灵活压缩历史视觉信息为紧凑令牌，同时保留短期连贯性和长期一致性。为生成与用户意图对应的语音和动作，我们提出推理-反应模块（RRM），该模块结合了状态循环策略与缓存切换机制。在多样化场景上的广泛实验结果显示，我们的方法在长时生成中实现了最先进的视觉一致性，同时能够实时进行复杂的用户-化身交互。

## Abstract
Recent diffusion-based models have enabled realistic audio-driven avatar generation in real-time streaming. However, existing approaches struggle to maintain visual temporal consistency and fail to explicitly perceive user intent in complex interactive streaming scenarios. To address these challenges, we propose InteractiveAvatar, a real-time infinite-streaming video generation framework that supports visually consistent avatar video generation and intent-aware interactions. With autoregressive distillation, InteractiveAvatar achieves real-time str-eaming generation of human avatars over arbitrarily long durations. For visual consistency, we introduce a Long-Short Visual Memory (LSVM) mechanism that flexibly compresses historical visual information into compact tokens, preserving both short-range coherence and long-term consistency. To generate avatars with speeches and actions aligned with user intent, we propose a Reasoning-Reaction Module (RRM), which incorporates a State-Cycling strategy and a Cache-Switching mechanism. Extensive experimental results over diverse scenarios demonstrate that our method achieves state-of-the-art visual consistency in long-duration generation, while enabling complex user-avatar interaction in real time.

- 因此，论文以“面向一致性与意图感知化身的实时流式视频生成”为核心问题，旨在同时解决时序稳定性和交互智能性，推动下一代沉浸式数字人发展。

### 2. 方法论
整体框架 InteractiveAvatar 以参考图像和用户语音为输入，通过**推理‑反应模块（RRM）** 生成意图对齐的语音与动作指令，再在**长短时视觉记忆（LSVM）机制**与自回归蒸馏的加持下，实时流式合成高一致性的化身视频。

#### 2.1 长短时视觉记忆（LSVM）
- **记忆压缩**：用轻量卷积‑注意力模型 \(C(\cdot)\) 将生成的视频潜变量帧 \(z_t\) 压缩为紧凑令牌 \(m_t = C(z_t)\)，时间压缩比为1，避免时间轴上的信息混合。
- **双缓冲区**：
  - **短期记忆** \(M_s\)：固定窗口大小 \(K\)，保存最近 \(K\) 帧的密集表达，保证局部连贯。
  - **长期记忆** \(M_l\)：固定容量 \(N\)，存储经动态选择后的代表性令牌，维持全局外观一致。
- **动态关键帧选择（DKFS）**：当帧从短期记忆弹出时，利用 SigLIP2 提取语义特征，计算若替换长期记忆某槽位后的全局冗余度下降幅度；仅当能使冗余度降低（即增加语义多样性）时才更新长期记忆，确保长期记忆紧凑且覆盖多样场景。
- 训练时，随机采样视频片段，短记忆范围直接用最近 \(K\) 帧，长记忆范围从更早帧中随机选若干帧，掩码部分帧后重建，迫使模型同时保持精细局部细节和稳健全局身份。

#### 2.2 推理‑反应模块（RRM）
- **状态循环策略**：LLM 接收用户文本 \(x_t\) 与上一稳定状态 \(S_{t-1}\)，输出动作状态（动作提示 \(p_{\text{act}}\)、回复音频 \(a_{\text{resp}}\)）和新的稳定状态提示 \(p_{\text{stable}}\)。生成阶段先按动作状态生成含动作与语音的视频，音频结束后切换至稳定状态提示，冻结人物姿态，避免非必要运动漂移。由此形成“推理‑反应‑稳定”的闭环。
- **缓存切换机制**：当动作提示发生切换时，仅对受影响的已生成块重新计算提示相关的 KV 缓存，避免重新生成整个历史，从而减少动作切换带来的延迟。

#### 2.3 自回归蒸馏适配
- 四阶段训练：
  1. 在大规模视听数据上训练**图像‑音频‑视频（AI2V）双向注意力基础模型**；
  2. **预训练记忆压缩模块** C(·)；
  3. **ODE 初始化**：改用块状因果注意力训练模型，逼近双向教师的 ODE 轨迹；
  4. **自强迫 DMD 蒸馏**：将扩散模型蒸馏为少步自回归生成器，配合 LSVM 实现高效实时推理。
